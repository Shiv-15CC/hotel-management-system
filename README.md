# 🏨 Hotel Management System

A JavaFX-based Hotel Management System built using **Java, JavaFX, Maven, and SQLite**.
This project helps manage hotel operations like customers, rooms, and billing.

---

## 🚀 Features

* Customer Management
* Room Management
* Booking System
* Billing System
* SQLite Database Integration
* JavaFX UI

---

## 🛠️ Tech Stack

* Java 25
* JavaFX
* Maven
* SQLite
* VS Code

---

## 📂 Project Structure

```
hotel-management-system
│── src
│── pom.xml
│── README.md
│── hotel-management.db
```

---

## ▶️ How to Run

### 1. Clone Repository

```
git clone https://github.com/Shiv-15CC/hotel-management-system.git
```

### 2. Navigate to Project

```
cd hotel-management-system
```

### 3. Run Project

```
mvn clean javafx:run
```
